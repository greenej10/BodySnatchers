Instructions

Left Arrow- Move Left
Right Arrow- Move Right
Space- Possess Animal
Alt- Depossess Animal
Z- Use Animal Action
	Monkey-Jump/Climb Vines
	Gorilla- Break Blocks
	Crocodile- Swim
Escape- Pause/Unpaused

CHEAT CODE

While Paused hit F1 to skip to the next level.
